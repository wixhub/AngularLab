* lint *

git checkout -b lint
//Switched to a new branch "lint"

Add airBnB to the project:

npm install eslint —-save-dev
npm install eslint-config-airbnb —-save-dev
npm install eslint-plugin-import eslint-config-airbnb-typescript --save-dev
ng add @angular-eslint/schematics

add "airbnb-typescript/base" to "extends" in .eslintrc.json
add "plugin:import/recommended" to "extends" in .eslintrc.json

add "@typescript-eslint/comma-dangle": "off" to "rules" in .eslintrc.json
add "import/named": "off" to "rules" in .eslintrc.json
https://eslint.org/docs/latest/rules/

ng lint
